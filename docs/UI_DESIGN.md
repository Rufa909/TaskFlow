 UI/UX Design Guidelines

## Color Scheme
- Primary: #3B82F6 (Blue)
- Success: #10B981
- Danger: #EF4444
- Background: Dark mode preferred (#0F172A)

## Layout
- Sidebar trái: Projects list
- Main area: Kanban board
- Top nav: Search, Notifications, Profile

## Components Style
- Task cards: Rounded, shadow, hover effect
- Modal: Centered, backdrop blur
- Responsive: Stack columns on mobile
