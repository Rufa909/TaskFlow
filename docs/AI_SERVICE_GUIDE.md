# AI Service Guide cho TaskFlow (Self-trained Marketing)

**Yêu cầu chính:**
- Không dùng bất kỳ API bên ngoài (OpenAI, Grok...)
- Tự train model
- Chuyên sâu Marketing

**Cách Minimal (khuyến nghị):**
- Dùng Flask + scikit-learn + sentence-transformers
- Train trên dataset Marketing tự tạo

Xem chi tiết ở phần dưới hoặc file train.py.
